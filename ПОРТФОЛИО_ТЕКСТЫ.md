# Тексты для портфолио FL.ru и карточка

## Две полноценные работы (функциональные лендинги)

Папки с готовыми сайтами:
- **portfolio-work-1** — Кофейня «Уголок» (тёмная тема, форма заказа стола)
- **portfolio-work-2** — Фотостудия «Свет» (светлая тема, форма бронирования зала)

Оба лендинга: навигация, блоки, форма заявки (после настройки Formspree отправляет на почту). Открой `index.html` в браузере — всё работает.

## Тексты для добавления работ на FL.ru

### Работа 1 — Кофейня «Уголок»

**Название:**
```
Лендинг кофейни — навигация, меню, форма заказа
```

**Описание:**
```
Полноценный одностраничный лендинг: фиксированная шапка с навигацией, герой, блок «О нас», карточки меню (популярные позиции), форма заказа стола/торта с отправкой заявки, контакты. Тёмная тема, адаптивная вёрстка. HTML, CSS, JS (форма). Подходит для кафе, ресторанов, услуг.
```

**Скриншот:** открой `portfolio-work-1/index.html` → сделай скрин (целиком или по блокам).

## Твоя карточка (сайт-визитка)

**Папка:** `my-card`

**Файл:** открой `my-card/index.html` в браузере.

На карточке:
- **Услуги:** одностраничный лендинг, сайт-визитка, адаптивная вёрстка, форма заявки.
- **Выполненные работы:** три карточки — «Кофейня», «Фотостудия», многостраничный сайт «Практик» (ссылки открываются в новой вкладке).
- **Контакты:** Telegram, email. Ссылку на профиль FL.ru замени на свою: открой `my-card/index.html` в редакторе, найди `Профиль на FL.ru` и вставь свой URL профиля.

**Чтобы формы на лендингах отправляли заявки на твою почту:** зайди на [formspree.io](https://formspree.io), создай бесплатный аккаунт и форму — тебе дадут ID (вид `xxxxxxxx`). В файлах `portfolio-work-1/index.html` и `portfolio-work-2/index.html` замени `YOUR_FORMSPREE_ID` на этот ID в атрибуте `action` формы.
